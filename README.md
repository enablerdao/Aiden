# Devin AI System

A comprehensive AI engineering system inspired by Devin, designed to assist with software development tasks.

## Features

- **Code Generation & Analysis**: Generate, review, and fix code
- **GitHub Integration**: Create and review PRs, manage repositories
- **Browser Integration**: Search documentation, test web applications
- **Shell Integration**: Execute commands, manage development environment
- **Development Environment**: FastAPI backend + React frontend

## Project Structure

- `/backend`: FastAPI server for AI processing and integrations
- `/frontend`: React + Vite web interface
- `/browser_integration`: Browser automation and web search capabilities
- `/shell_integration`: Shell command execution and environment management
- `/github_integration`: GitHub API integration for repository management
